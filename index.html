The Game

Parietals Break is a decision-based game where you, the palyer, live through
a day and evening at Dotre Name University.

Design

We used Bitmojis and free-use images to create the game. This is probably not illegal, but what do we know? We're not lawyers.

Development

The game is written in Python and makes use of the Pygame module. It is an open source project on github, so anyone can contribute.
